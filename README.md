 # Cloud-Based AI-Driven Self-Healing WAF

This project implements a cloud-based Web Application Firewall (WAF) enhanced with AI-based anomaly detection.
The system detects malicious traffic patterns and automatically applies mitigation actions.

Tech Stack:
- Cloud: AWS
- WAF: NGINX + ModSecurity
- AI: Python + Isolation Forest
- Logging: ELK Stack / CloudWatch
- App: Flask (demo vulnerable app)
